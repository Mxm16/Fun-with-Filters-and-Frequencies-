1.3 Derivative of Gaussian (DoG) Filter

The unsmoothed gradient images are noisy. Using a Gaussian filter G, we can smooth the image before applying finite differences. One way to create a 2D Gaussian is by taking the outer product of a 1D Gaussian with its transpose (e.g., via cv2.getGaussianKernel()).

Compared with unsmoothed gradients, these smoothed gradients are less noisy: small texture variations are suppressed while main edges become clearer. Derivative of Gaussian (DoG) filters combine smoothing and differentiation in a single convolution step.

Applying the DoG filters directly to the original image produces the same result as smoothing first and then computing derivatives.
